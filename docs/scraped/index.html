시작하기

시작하기

디자인 시스템 구축을 위한 모든 자료가 준비되어 있어요. 지금 바로 최신 정보를 확인하고, 프로젝트를 시작해 보세요.

  • 튜토리얼 디자인 프로세스를 쉽게 배우세요

    단계별로 디자인 원칙을 익힐 수 있도록 안내합니다. 실제 사례를 통해 실용적인 노하우를 쌓아보세요.

  • 리소스 다양한 자료를 활용하세요

    디자인에 필요한 다양한 자료와 도구를 제공합니다. 템플릿, 아이콘, 색상 팔레트 등, 프로젝트에 필요한 모든 자료를 손쉽게 찾고 활용할 수 있습니다.

  • 디자인 토큰 일관성 있는 디자인을 위한 필수 요소

    디자인의 기본 요소를 코드로 정의하여 일관성을 유지할 수 있도록 돕습니다. 디자인 언어를 통일하고, 효율적인 스타일 가이드를 구축하세요.

  • 튜토리얼 코드로 디자인을 구현하는 방법 배우기

    디자인 원칙을 코드로 구현할 수 있도록 단계별로 안내합니다. 실습 중심의 자료를 통해 필요한 기술을 익히고, 프로젝트에 바로 적용해 보세요.

  • 리소스 다양한 자료를 활용하세요

    API 문서, 코드 샘플, 라이브러리와 같은 유용한 자료를 제공합니다. 효율적인 개발을 위해 필요한 모든 도구를 한 곳에서 찾아보세요.

  • 튜토리얼 효율적인 활용 방법 배우기

    시스템을 효과적으로 관리하고 최적화할 수 있도록 단계별로 안내합니다. 콘텐츠를 통해 운영 과정에서의 핵심 기술을 익혀보세요.

  • 리소스 필요한 자료를 제공받으세요

    가이드 적용을 위한 체크리스트를 제공합니다. 자료를 통해 보다 체계적이고 효과적인 운영을 지원받으세요.

주요 구성

디자인 시스템의 뼈대를 이루는 구성 요소에 대해 알아볼까요?

  • 디자인 원칙

    디지털 정부서비스 UI/UX의 방향성과 설계 기준이 되는 상위 원칙

  • 디지털 포용

    모든 국민이 동일한 서비스를 경험할 수 있도록 디지털 소외 계층을 위한 가이드를 제공

  • 스타일

    컴포넌트, 기본 패턴을 시각적으로 일관성 있게 표현하기 위한 규칙

  • 컴포넌트

    사용자 인터페이스의 가장 작은 단위로 일관성 있게 사용되는 공통 요소에 대한 가이드

  • 기본 패턴

    핵심 과업을 수행하는데 반복적으로 사용되는 사용자 인터페이스 집합에 대한 가이드

  • 서비스 패턴

    핵심 과업에 대한 사용자 여정 기반의 사용자 경험 설계 가이드
