# 시작하기

## 설치

```bash
# npm
npm install krds-community-react

# pnpm
pnpm add krds-community-react

# yarn
yarn add krds-community-react
```

## CSS 설정

프로젝트 루트(또는 레이아웃)에서 스타일을 import합니다:

```tsx
// app/layout.tsx (Next.js App Router)
import "krds-community-react/styles";

export default function RootLayout({ children }) {
  return (
    <html lang="ko">
      <body>{children}</body>
    </html>
  );
}
```

```tsx
// main.tsx (Vite/CRA)
import "krds-community-react/styles";
import { createRoot } from "react-dom/client";
import App from "./App";

createRoot(document.getElementById("root")!).render(<App />);
```

## 기본 페이지 구조

KRDS 공공 서비스 페이지의 기본 구조:

```tsx
import {
  Masthead,
  Header,
  SkipLink,
  Breadcrumb,
  BreadcrumbItem,
  Footer,
  Identifier,
} from "krds-community-react";

function Page() {
  return (
    <>
      <SkipLink targetId="main-content" />
      <Masthead />
      <Header title="서비스명" />

      <Breadcrumb>
        <BreadcrumbItem home href="/">홈</BreadcrumbItem>
        <BreadcrumbItem href="/services">서비스</BreadcrumbItem>
        <BreadcrumbItem current>현재 페이지</BreadcrumbItem>
      </Breadcrumb>

      <main id="main-content">
        {/* 본문 콘텐츠 */}
      </main>

      <Footer>
        <Identifier org="행정안전부" />
      </Footer>
    </>
  );
}
```


## 모달 사용

```tsx
import { Modal, ModalTrigger, ModalContent, ModalHeader, ModalBody, ModalFooter, ModalClose, Button } from "krds-community-react";

function ConfirmDialog() {
  return (
    <Modal>
      <ModalTrigger render={<Button variant="tertiary">삭제</Button>} />
      <ModalContent>
        <ModalHeader>삭제 확인</ModalHeader>
        <ModalBody>
          선택한 항목을 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다.
        </ModalBody>
        <ModalFooter>
          <ModalClose className="krds-btn medium tertiary">취소</ModalClose>
          <Button variant="primary" size="medium">삭제</Button>
        </ModalFooter>
        <ModalClose />
      </ModalContent>
    </Modal>
  );
}
```

## Next.js App Router

모든 컴포넌트에 `'use client'` 지시어가 포함되어 있어 Server Component 내에서 직접 import 가능합니다:

```tsx
// app/page.tsx (Server Component)
import { Button } from "krds-community-react";

export default function Page() {
  return <Button variant="primary">서버 컴포넌트에서 사용</Button>;
}
```

인터랙티브 로직이 필요한 경우 Client Component로 분리:

```tsx
// app/components/LoginForm.tsx
"use client";
import { useState } from "react";
import { TextInput, Button } from "krds-community-react";

export function LoginForm() {
  const [email, setEmail] = useState("");
  return (
    <form>
      <TextInput label="이메일" value={email} onChange={(e) => setEmail(e.target.value)} />
      <Button type="submit">로그인</Button>
    </form>
  );
}
```
